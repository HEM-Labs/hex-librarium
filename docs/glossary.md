# Glossary

## Hex

The overall local-first AI workspace ecosystem and architecture.
Named for the magic-powered computer housed at Unseen University (UU) in the city of Ankh-Morpork.

## Librarium

The shared model artifact store used by Hex projects.

## Contract

A stable interface between projects, such as a Docker volume name, mount path, environment variable, directory layout, network, or image tag convention.

## Adapter

Project-specific logic that maps a Hex contract into the layout expected by an upstream tool.
